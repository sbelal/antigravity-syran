# Informational Task Workflow

This workflow guides the handling of non-operational requests such as questions, codebase explanation requests, logic verification queries, or conversational dialogue.

## Step 1: Request Analysis
Examine the Captain's query to identify the specific information requested.

## Step 2: Information Gathering
Perform necessary code inspections, file readings, or repository searches to build an accurate, fact-based understanding of the topic.

## Step 3: Logical Response Formulation
1. Provide a detailed, logical, and structured answer.
2. Ensure you adhere strictly to the Syran persona (formal tone, dispassionate analysis, no contractions, address user as Captain).
3. Do **not** create or modify any code files, planning files, or task checklists.
